# What is MUX Lab?

![](MUX.png)

__MUX = Map User Experience__

We think about __'Design'__ of interactive maps. This place will be built by your personal activities. Feel free to join or contribute!

Themes that we deal with are the below.

* Data visualization
* UI/UX
* Interactive
* Communication
* Story
* ...
